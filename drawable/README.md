# Drawable
![ICON](https://raw.githubusercontent.com/AlexMofer/ProjectX/master/drawable/icon.png)

实现特定功能的Drawable合集。
## 要求
minSdkVersion 4
## 引用
```
dependencies {
    ⋯
    compile 'am.drawable:drawable:1.4.0'
    ⋯
}
```
## 详情
- **LineDrawable**

    横线图片，主要是底色为透明或半透明色时有用，为不透明时，通过layer-list即可实现。
- **CornerDrawable**
    
    尖角框，使用该Drawable时，会改变View的Padding值。
- **TextDrawable**
    
    文字图片。
